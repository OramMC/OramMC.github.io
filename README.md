# OramMC's Academic Homepage

🌌 Personal academic website featuring gravitational wave and pulsar research.

## 🚀 Quick Start

This website is deployed using GitHub Pages. Visit: https://orammc.github.io

## 📁 Project Structure

```
OramMC.github.io/
├── index.html          # Main HTML file
├── css/
│   └── style.css       # Stylesheet (Deep Space theme)
├── js/
│   └── main.js         # JavaScript functionality
├── images/
│   ├── profile.jpg     # Your profile photo
│   ├── paper1.png      # Paper thumbnails
│   └── gallery/        # Gallery photos
├── assets/
│   ├── cv.pdf          # Your CV
│   └── slides/         # Weekly report slides
└── README.md           # This file
```

## ✏️ How to Update

### Update Personal Info
Edit `index.html` and change:
- Name and title in the sidebar
- Email and social links
- About me section content

### Add New Publication
Add a new `<div class="pub-item">` block in the Publications section.

### Upload Weekly Reports
1. Add PDF files to `assets/slides/`
2. Add corresponding links in the Weekly Reports section

### Add Gallery Photos
1. Add photos to `images/gallery/`
2. Add corresponding `<div class="gallery-item">` blocks

## 🎨 Theme
